CSS es un lenguaje de estilos utilizado para definir la presentación de documentos HTML. Permite modificar colores, fuentes, márgenes y estructuras visuales de una página web.

Existen diferentes formas de aplicar CSS: estilos en línea, internos y externos. La mejor práctica es utilizar hojas de estilo externas para mantener el código organizado.

CSS también permite la creación de diseños responsivos mediante el uso de media queries, lo que facilita la adaptación de una web a diferentes dispositivos y tamaños de pantalla.
